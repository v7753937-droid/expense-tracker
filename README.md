# 💰 Учёт трат

Приложение для учёта личных финансов с современным интерфейсом на русском языке.

## 🚀 Возможности

- **Добавление трат** — ввод суммы, категории, описания и даты
- **8 категорий** — еда, транспорт, развлечения, покупки, здоровье, коммунальные, образование, другое
- **Фильтрация** — по категории и дате
- **Удаление трат** — с подтверждением
- **Аналитика** — общая сумма, количество трат, средняя трата
- **Разбивка по категориям** — с визуальными диаграммами
- **Последние траты** — быстрый просмотр последних 5 трат
- **Сохранение данных** — все данные сохраняются в localStorage браузера

## 📦 Установка

Приложение не требует установки зависимостей. Просто скачайте файлы и откройте `index.html` в браузере.

## 🎯 Как использовать

1. Откройте файл `index.html` в любом современном браузере
2. На вкладке "Траты" заполните форму:
   - Сумма (в рублях)
   - Категория (выберите из списка)
   - Описание (необязательно)
   - Дата (по умолчанию сегодня)
3. Нажмите "Добавить"
4. Трата появится в списке ниже
5. Используйте фильтры для поиска трат по категории или дате
6. Перейдите на вкладку "Аналитика" для просмотра статистики

## 💾 Хранение данных

Все данные сохраняются в `localStorage` браузера. Данные не удаляются при закрытии браузера, но будут потеряны если очистить данные браузера.
